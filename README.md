# Project Overview
The P2P Tutoring Platform is a full-stack web application that enables students to:
Post questions
Answer peer queries
Learn collaboratively in a structured environment

# Tech Stack
Frontend: React.js
Backend: Node.js, Express.js
Database: MongoDB
Authentication: JWT (JSON Web Tokens)

# Key Features
User authentication (Register/Login)
Post and view questions
Answer questions
RESTful API architecture
Modular and scalable code structure

# Testing
